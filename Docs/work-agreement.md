The group members will decide on all the tasks, and members can choose the tasks they are most comfortable with or are willing to put in the time to research and complete. Difficult tasks can be shared among more than 1 member

## 7. Conflict Resolution & Unmet Expectations

If conflict arises or expectations are not met, we agree to follow this process in order:

1. All group members have a meeting and discuss the issue with the member in question
2. The responsible member(s) are given a reasonable deadline to finish the task
3. The rest of the group members can constantly be checking in to see if the task is being done; if not, then the instructor may have to be involved

*(Examples: direct conversation → team discussion → instructor escalation)*

**How will concerns be raised respectfully?**  
We will set expectations for each member, and if they are not met, we will raise the issue during the next meeting. Let the member know the issue is affecting the rest of the group, and if no actions are taken then we can decrease their points in the peer review process

## 8. Responsible GenAI Use

As a team, we have discussed the responsible and ethical use of Generative AI tools (e.g., ChatGPT, Copilot) for this project. We agree on the following shared norms:

**Allowed uses of GenAI in this project include:**
- Debugging code – not just the affected functions
- Brainstorming ideas on the presentation/UI stage
- Improving code quality and efficiency

**Disallowed uses of GenAI in this project include:**
- Copying the whole code and pasting it into the project without understanding the code
- Generating documents totally from GenAI
